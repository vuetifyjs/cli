# create-vuetify

The easiest way to start a new Vuetify project.

## Usage

With **npm**:
```bash
npm create vuetify
```

With **pnpm**:
```bash
pnpm create vuetify
```

With **yarn**:
```bash
yarn create vuetify
```

With **bun**:
```bash
bun create vuetify
```
