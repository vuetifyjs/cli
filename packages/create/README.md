# create-vuetify

Create a new Vuetify project.

## Usage

```sh
# npm
npm create vuetify

# pnpm
pnpm create vuetify

# yarn
yarn create vuetify

# bun
bun create vuetify

# deno
deno run -A npm:create-vuetify
```

## Status

Work in progress.

## Development

This package is part of the Vuetify CLI monorepo. From the repo root:

- Install deps: `pnpm install`
- Build: `pnpm -F create-vuetify run build`
