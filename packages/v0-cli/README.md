# @vuetify/cli-v0

Vuetify0 CLI utilities and commands for Vuetify.

## Usage

```sh
# npm
npx @vuetify/cli-v0

# pnpm
pnpm dlx @vuetify/cli-v0

# yarn
yarn dlx @vuetify/cli-v0

# bun
bunx @vuetify/cli-v0

# deno
deno run -A npm:@vuetify/cli-v0
```

## Status

Work in progress.

## Development

From the monorepo root:

- Install deps: `pnpm install`
- Build: `pnpm -F @vuetify/cli-v0 run build`
